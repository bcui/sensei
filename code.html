Reporting Bugs

We use Jira as issue tracker.

Bug reports are incredibly helpful, so take time to report bugs and request features in our ticket tracker. We’re always grateful for patches to Sensei’s code.

Bug reports with attached patches will get fixed far quickly than those without any.

Getting the code

Sensei is hosted on Github using the Git version control system. You can get the code with: 

git clone git://github.com/javasoze/sensei.git

Building the code

Sensei uses Apache Maven to build.

First start ZooKeeper by going to Zookeeper install directory and do: 

./bin/zkServer.sh start

Go into top-level directory for Sensei

mvn package

The artifact will be built to the target directory.

Testing

You can run the unit test with the following command: (Make sure Zookeeper is running)

mvn test
